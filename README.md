# Real-Engine-Analysis
MATLAB-based Real Engine Analysis GUI developed for propulsion studies. The tool analyzes real turbojet and turbofan cycles considering component efficiencies and non-ideal conditions. It calculates thrust, SFC, and efficiencies via an interactive interface. Available as open-source code and standalone installer.

The main purpose of this project is to make propulsion cycle analysis more interactive and easier to understand for aerospace engineering students.
